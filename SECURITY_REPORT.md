## Methodology

All data was extracted using `solscan-cli --power-map`, which:

1. Queries each program's account via `getAccountInfo`
2. Identifies BPF Upgradeable Loader programs
3. Decodes the programdata account address from program account data
4. Extracts the upgrade authority from the programdata account
5. Profiles each authority wallet (balance, transaction count)

**No private data was accessed.** All information is publicly available on-chain.
